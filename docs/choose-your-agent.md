# Choose Your Agent

This starter kit works with Claude Code and Codex.

## Use Claude Code when

- You want slash commands like `/start`.
- You want Claude Code plugins and local Claude skills.
- You like a terminal-first coding flow inside VS Code.

First message:

```text
/start
```

## Use Codex when

- You want Codex to read the whole project context.
- You want a project-agent workflow based on `AGENTS.md`.
- You want the same starter kit without depending on Claude Code slash commands.

First message:

```text
I am new. Read this starter kit and help me set it up one question at a time. Do not change files until I approve.
```

The exact interface can change over time. The idea stays the same: one starter
kit, two agent paths.
