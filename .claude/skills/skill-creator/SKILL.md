# Skill Creator

Use this skill when the user repeats the same instructions or wants a reusable
AI workflow.

## Interview

Ask one question at a time:

1. What task should this skill help with?
2. What should trigger the skill?
3. What steps should the AI always follow?
4. What should the AI avoid?
5. What does "done" look like?

## Output

Create a draft skill with:

- Name
- Description
- Trigger examples
- Step-by-step workflow
- Output format
- Done criteria

## Rule

Do not create a skill for a one-off task. Create a skill only when the process
will be reused.
